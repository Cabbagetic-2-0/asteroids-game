## Features:
**Responsive Canvas:**
Automatically resizes to fit your screen.

**Touch Controls:**
Fully functional mobile controls that hide on desktop.

**Visuals:**
CRT Scanline overlay for that retro TV feel.
Geometric shapes (Triangular ship, Square asteroids) rendered via code (no images).
Particle explosions and thrust trails.
"Press Start 2P" pixel font.

How to play? go to [how-to-play.md](https://github.com/Cabbagetic-2-0/asteroids-game/blob/main/how-to-play.md)
